# Signal – Mechanics Service Platform

## 🚀 Overview
Signal is a service platform that connects users with nearby mechanics in real-time. Users can request help and get matched with available mechanics based on location.
website:https://signal-rho-three.vercel.app/
## ⚙️ Tech Stack
- React
- Next.js
- Firebase
- Maps API

## ✨ Features
- Real-time service requests
- Location-based mechanic matching
- User authentication
- Service request tracking
- Interactive map integration

## 🧠 Architecture
User sends request → Firebase stores request → system matches nearest mechanics → mechanics respond → status updates in real time.
